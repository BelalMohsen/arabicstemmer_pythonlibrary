# Arabic stemmer python library
Python library for Assem's Arabic Stemmer

## Install

```sh
   $ sudo pip install arabicstemmer
```

## Usage 

```python

    >>> from arabicstemmer import ArabicStemmer
    >>> stemmer = ArabicStemmer()
    >>> stemmer.stemWord(u"المسالمون")
    'مسالم'    
```
